# Sistema de Recomendação de Filmes — OIAA (Etapa 1) 🎬

Este repositório contém a solução padronizada para o desafio de Machine Learning (Etapa 1) da Olimpíada de IA Aplicada. O objetivo do modelo é prever as avaliações (ratings) de filmes feitas por usuários utilizando uma abordagem clássica de filtragem colaborativa.

## 📁 Estrutura do Repositório

* `ML_E1_padronizado.ipynb`: Notebook oficial do desafio estruturado conforme o padrão da competição.
* `requirements.txt`: Lista de dependências e bibliotecas necessárias para a execução.
* `README.md`: Instruções de instalação e execução.

## 🚀 Como Executar o Projeto

### Pré-requisitos
Certifique-se de ter o Python 3.8+ instalado em sua máquina.

### Passo 1: Clonar o repositório ou baixar os arquivos
Baixe os arquivos deste repositório para a sua máquina local.

### Passo 2: Instalar as dependências
Abra o seu terminal na pasta do projeto e execute o comando abaixo para instalar as bibliotecas necessárias:
```bash
pip install -r requirements.txt
